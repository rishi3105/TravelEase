This project is a Travel Website built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It allows users to book hotels and create customized itineraries powered by the Gemini API.
<br />
<br />Features
<br />Hotel Booking: Browse and book hotels from various destinations.
<br />Customized Itinerary: Generate a tailored travel itinerary based on user preferences using the Gemini API.
<br />User Authentication: Register and log in to save bookings and itineraries.
<br />Responsive Design: Optimized for desktop and mobile devices.
<br />Secure Payments: Secure hotel booking with payment integration (if applicable).
<br />
<br />Tech Stack
<br />Frontend: React.js (with React Router for navigation)
<br />Backend: Node.js, Express.js
<br />Database: MongoDB
<br />API Integration: Gemini API for generating itineraries
<br />Authentication: JWT for secure user sessions
<br />State Management: React Context API or useReducer
<br />
<br />Installation Prerequisites
<br />Node.js and npm installed
<br />MongoDB installed and running
<br />Gemini API key
